What is @media query and how can we use it?

Media queries are used in CSS to apply different styles to a web page based on the characteristics of the device that is being used to view the page, such as screen size, orientation, and resolution.

In this example, the @media rule is used to specify that the following styles should only apply to screens with a maximum width of 767 pixels. The screen keyword specifies that the media type is a screen, and the (max-width: 767px) condition specifies the maximum screen width. You can use other conditions in media queries, such as min-width, orientation, aspect-ratio, and resolution, to create more targeted styles for different devices.

What is responsive website?

Responsive website is a website that is designed to adapt and adjust its layout and content to fit the screen size and orientation of the device being used to view it. The goal of a responsive website is to provide a consistent and optimal user experience across a range of devices, from desktops and laptops to tablets and smartphones.

Responsive design techniques typically involve using flexible grids, images, and media queries to adjust the layout and content of a web page based on the screen size and resolution of the device.
